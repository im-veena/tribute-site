# Tribute Website – Indian Leaders

This project is a simple **HTML & CSS tribute website** created to honor great Indian leaders and showcase basic front-end development skills.

The website focuses on clean layout, readable typography, and structured content, making it suitable as a beginner project and portfolio addition.

## Project Features

- Responsive and clean layout  
- Tribute cards for multiple Indian leaders  
- Images with proper alignment and styling  
- Well-structured sections (About, Timeline, Quotes)  
- Beginner-friendly HTML and CSS code  

## Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling and layout
